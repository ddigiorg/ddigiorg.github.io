## Instructions

Create a virtual environment

```bash
python -m venv <path>
```

Activate (Linux)

```bash
source <path>/bin/activate
```

Deactivate (Linux)

```bash
deactivate
```

Activate (Windows)

```bash
<path>\Scripts\activate.bat
```

Deactivate (Windows)

```bash
<path>\Scripts\deactivate.bat
```
